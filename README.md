# xx-ui
一个简约的UI框架

### 介绍
此框架意在学习之用，希望能将它做好。谢谢！

简单组件：icon、button、button-group

### 安装
```
npm install xx-ui
```

#### 注意
css，需要先改为border-box


### 快速上手
```
import {Button,ButtonGroup,Icon} from 'xx-ui'
export default {
    components: {
        'x-button':Button,
        'x-button-group':ButtonGroup,
        'x-icon':Icon
    }
}
```
